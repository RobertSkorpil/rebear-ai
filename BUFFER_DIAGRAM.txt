PATCH BUFFER FORMAT DIAGRAM
============================

Single Patch Example:
┌─────────────────────────────────────────────────────────────────────┐
│                        PATCH BUFFER (17 bytes)                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌────────────────────────────────────────────────┐                 │
│  │     PATCH HEADER 0 (8 bytes)                   │                 │
│  ├─────┬──────────┬────────┬─────────────┐       │                 │
│  │ 0x80│0x00 10 00│00 08   │00 09         │       │                 │
│  │     │          │        │              │       │                 │
│  │STORE│ ADDRESS  │ LENGTH │BUFFER_DATA   │       │                 │
│  │(ena)│(0x001000)│  (8)   │  (offset 9)  │       │                 │
│  └─────┴──────────┴────────┴─────────────┘       │                 │
│                                                    │                 │
│  ┌─────┐                                          │                 │
│  │ 0x00│  TERMINATOR (1 byte)                     │                 │
│  └─────┘                                          │                 │
│                                                    │                 │
│  ┌──────────────────────────────────────────────┐ │                 │
│  │ 0xFF FF FF FF FF FF FF FF                    │ │                 │
│  │                                               │ │                 │
│  │ PATCH DATA 0 (8 bytes)                       │ │                 │
│  └──────────────────────────────────────────────┘ │                 │
│                                                    │                 │
└────────────────────────────────────────────────────────────────────┘


Multiple Patches Example:
┌─────────────────────────────────────────────────────────────────────┐
│                        PATCH BUFFER (33 bytes)                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌────────────────────────────────────────────────┐                 │
│  │     PATCH HEADER 0 (8 bytes)                   │                 │
│  ├─────┬──────────┬────────┬─────────────┐       │                 │
│  │ 0x80│0x00 10 00│00 08   │00 11         │       │  Enabled        │
│  └─────┴──────────┴────────┴─────────────┘       │                 │
│                                                    │                 │
│  ┌────────────────────────────────────────────────┐                 │
│  │     PATCH HEADER 1 (8 bytes)                   │                 │
│  ├─────┬──────────┬────────┬─────────────┐       │                 │
│  │ 0x00│0x00 20 00│00 08   │00 19         │       │  Disabled       │
│  └─────┴──────────┴────────┴─────────────┘       │                 │
│                                                    │                 │
│  ┌─────┐                                          │                 │
│  │ 0x00│  TERMINATOR (1 byte)                     │                 │
│  └─────┘                                          │                 │
│                                                    │                 │
│  ┌──────────────────────────────────────────────┐ │                 │
│  │ 0x01 02 03 04 05 06 07 08                    │ │  Patch 0 data  │
│  └──────────────────────────────────────────────┘ │  @ offset 17   │
│                                                    │                 │
│  ┌──────────────────────────────────────────────┐ │                 │
│  │ 0xAA BB CC DD EE FF 11 22                    │ │  Patch 1 data  │
│  └──────────────────────────────────────────────┘ │  @ offset 25   │
│                                                    │                 │
└────────────────────────────────────────────────────────────────────┘


Offset Calculation:
┌────────────────────────────────────────────────────────┐
│  For N patches (N ≤ 8):                                │
│                                                         │
│  Headers section:     N × 8 bytes                      │
│  Terminator:          1 byte                           │
│  Data section:        N × 8 bytes                      │
│                                                         │
│  First patch data offset:  (N × 8) + 1                 │
│  Patch[i] data offset:     (N × 8) + 1 + (i × 8)       │
│  Total buffer size:        (N × 8) + 1 + (N × 8)       │
│                          = (N × 16) + 1                 │
│                                                         │
│  Examples:                                              │
│    1 patch:  17 bytes, first data at offset 9          │
│    2 patches: 33 bytes, first data at offset 17        │
│    3 patches: 49 bytes, first data at offset 25        │
│    4 patches: 65 bytes, first data at offset 33        │
│    8 patches: 129 bytes, first data at offset 65       │
│                                                         │
│  ⚠️ HARDWARE LIMIT: Maximum 8 patches per buffer       │
│     (9th position reserved for terminating header)     │
│                                                         │
└────────────────────────────────────────────────────────┘


Field Encoding (Big-Endian):
┌────────────────────────────────────────────────────────┐
│  STORED (1 byte):                                       │
│    0x80 = Enabled                                       │
│    0x00 = Disabled or Terminator                        │
│                                                         │
│  ADDRESS (3 bytes):                                     │
│    [HIGH] [MID] [LOW]                                   │
│    Example: 0x001000 → 0x00 0x10 0x00                  │
│                                                         │
│  LENGTH (2 bytes):                                      │
│    [HIGH] [LOW]                                         │
│    Always: 0x00 0x08 (8 bytes)                         │
│                                                         │
│  BUFFER_DATA (2 bytes):                                 │
│    [HIGH] [LOW]                                         │
│    Example: offset 9 → 0x00 0x09                       │
│             offset 17 → 0x00 0x11                      │
│             offset 256 → 0x01 0x00                     │
│                                                         │
└────────────────────────────────────────────────────────┘
